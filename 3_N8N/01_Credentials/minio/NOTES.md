# Prerequisite

> WARNING!! If running N8N in docker, and MSSQL in docker, do not use `localhost`, but rather real IP address ex. `192.168.2.114`.

How to find a IP address:

Windows - `ipconfig`
Linux/MacOS - `ifconfig`

# Notes

S3 Endpoint: `http://<IP_ADDRESS>:9000`
Access Key ID: `admin`
Secret Access Kye: `password123`
Force Path Style: TRUE
Ignore SSL Issues: TRUE

S3 Credentials: https://docs.n8n.io/integrations/builtin/credentials/s3/?utm_source=n8n_app&utm_medium=credential_settings&utm_campaign=create_new_credentials_modal
