# Prerequisite

> WARNING!! If running N8N in docker, and MSSQL in docker, do not use `localhost`, but rather real IP address ex. `192.168.2.114`.

How to find a IP address:

Windows - `ipconfig`
Linux/MacOS - `ifconfig`

# Notes

No username
No password
Url = `http://<IP_ADDRESS>:9200`
